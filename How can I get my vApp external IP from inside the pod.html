An EchoIP container is running in the lab environment. This service provides the visitor with the IP address. In the case of accessing the service from within a vPod, you will see your "External IP" address - the address needed for RDP to your pod from a Horizon Desktop.

EchoIP Address: http://my-ip.livefire.lab 

Usage via curl: 

curl http://my-ip.livefire.lab/ip

This service is ONLY available within the Lab Environment.

There is a public instance of this same code running at: https://ifconfig.co 

A properly configured ControlCenter should display the Public IP Address as shown in the following Screenshot:

Please DO NOT rely on your Public IP being consistently 4.1-4.254 as the overall range for Public IPs is:

172.5.4.5 -> 172.25.7.254

This range is provided to vCD and it randomly chooses which IP it assigns to the Edge serving the Public IP of your pod.
