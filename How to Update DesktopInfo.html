The DesktopInfo utility has been used for many years to provide information in the bottom-right corner of the vPods ControlCenter/Main Console. In 2022, we have an updated version available (3.6.0) that includes options for showing multiple timezones. Please follow the steps below to update your vPod's Main Console

1. Clean out Old DesktopInfo version

Perform the steps below from the Main Console inside your pod:

  1. Right-click on your task bar and launch the Task Manager
  2. Locate the DesktopInfo process and end it
  3. Open Windows Explorer to c:\DesktopInfo
  4. Delete the contents of the folder - if DesktopInfo.exe does not delete, please re-visit steps 1 and 2

2. Get Updated binary, ini, and supporting files

Perform the steps below from the Main Console inside your pod:

  1. Open Windows explorer and browse to \\172.25.0.12\attendee-share\dtinfo-360
    When prompted for credentials:
    Username: hzn-fs-01\vmware
    Password: VMware1!
  2. Copy ALL FILES from \\172.25.0.12\attendee-share\dtinfo-360
  3. Paste the copied files into the empty c:\DesktopInfo folder
  1. DELETE the c:\DesktopInfo\desktopinfo.ini file - this .ini file is for the Horizon Desktops, not for the vPods (not shown)
  2. RENAME the c:\DesktopInfo\desktopinfo-for-vpods.ini file to desktopinfo.ini
  1. Double-Click the c:\DesktopInfo\DesktopInfo.exe file
    NOTE: You may also reboot if you wish. The desktopinfo.exe file should already be configured in Main Consoles to auto-run upon login

3. Verify DesktopInfo

After a reboot or double-clicking the DesktopInfo.exe file, the information should show as seen in the screenshot above.

4. Need Changes?

The layout and information shown is determined by the settings in c:\DesktopInfo\desktopinfo.ini

  1. If you need to change the title shown (IE: ISEC Livefire 2.0):
    - Edit the end of line 48
  2. If you want to change the HOST value to something other than "Main Console"
    - Edit the end of line 49
  3. If the Timezones don't align (due to Daylight savings for example), those are offsets in +/- number of hours
    - Edit the end of lines 67, 68, and 69 as needed

 NOT SHOWN: Font Size

If you wish to adjust the font size (currently set to 14), edit line number 23. Size 13 looks good too, but 12 and below doesn't look so great...
