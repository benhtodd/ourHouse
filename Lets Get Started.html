If "Do not verify server identity certificates" is selected, then the box does not need to be checked.

Access VMware Cloud Services Portal
  1. Open Chrome and click the VMware Cloud Services link
  2. Log in using your email address, reset your password and complete the login

3. Select the Livefire Labs Organization listed under your name at the top-right of the window to reveal available VMware Cloud Services in this "org".

Create an API-Token
  1. Click the drop down menu at the top right of your screen
  2. Click My Account
  3. Click API Tokens
  4. Click GENERATE TOKEN
  5. Name the token your first initial + surname-admin
  6. Set the TTL to five days
  7. Click Generate

DO NOT CLOSE THIS WINDOW UNTIL YOU COPY THE TOKEN.  

For security reasons, after you generate the token, we only display the name of the token on the API Tokens page and NEVER the token credentials.  This is the FIRST, LAST, and ONLY time you will get to see your token presented to you in the interface.  COPY the credential and create a file to save it in as per the instructions below.  Also DOWNLOAD an image of your token NOW to act as a backup just in case.

  1. Click COPY
  2. Open a terminal window and type code api-token
  3. Type the name of your token, and then on a new line, paste the token
  4. Save the file
