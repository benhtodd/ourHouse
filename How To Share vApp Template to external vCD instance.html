As we want to share only specific vApp templates on demand we have created a new vCD organization called "Livefire-Template-Sync". All instructors have "Catalog author" role assigned in that organization. The user used for copying the templates has a "Read-Only" role. This way the requestor, who wants to copy the template, can only download it or sync it using the CMS tool.

1. Copy the template to the "Livefire-Template-Sync"organization

Log in to the "Livefire-Template-Sync" organization with your credentials using the following URL https://vcd.livefire.dev/tenant/Livefire-Template-Sync/ . Navigate to the Libraries tab and select the vApp template you want to share and copy it to the "Livefire-Template-Sync-Catalog"

 

Once the task finishes, the template is ready for sync/download.

Make sure that the VMs of your template are consolidated as this will provide the best chance of a successful copy to the Livefire-Template-Sync Org

2. Copy the template using the CMS tool

Information about the tool and how to use it can be found here https://confluence.eng.vmware.com/x/5LADDw . Please note you need to be connected to the corporate VPN (GlobalProtect) to access the tool and its documentation.

For sharing credentials there are 2 options:

1. Use the 1Password built-in feature.

2. The corporate "PrivateBin" can be used - https://securesrv-prod-3.vmware.com/ where it can be specified to burn the credentials after being read once.

Navigate to https://oc-cms.oc.vmware.com:8443/ui/ and select "template" from the drop down menu on the top. On the "Source Parameters" login with the credentials from the Livefire 1Password "Livefire-Labs" vault.

Make sure to enter "livefire_template_sync@Livefire-Template-Sync" as a username since we need to specify the org the user belongs to.

The "Target Parameters" should be provided by whoever wants to sync the template.

Once logged in organization and catalog populate automatically (organization is already included in the username and the user has access to only one catalog). Select the transfer settings at the bottom as per your requirement and hit "Request Transfer". The progress is shown and once finished you should see that at the end of the log.

3. Download the template

Log in to the org URL using the "livefire_template_sync" user, go to Libraries -> vApp Templates, select the template and click "Download". A new window will pop up, select the options as per your needs and click OK. The download will start.

The actual download will start after the template has been exported from the vCenter in the backend. This operation is also know as "enabling download" in vCD.
