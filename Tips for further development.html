Creating new scripts for newly developed lab tasks

When you create a new vApp with new tasks, the Lab Checkers scripts (Lab-CNS-Full-v6.ps1) needs to be updated with new API calls and the things that need to be checked.

I suggest to create simple scripts that does checking on a task basis, then combine multiple tasks into a script so you can check on a per lab basis, and if all the lab scripts are working (consisting of multiple tasks) then create one large script that checks all labs. I have used the same strategy.
