On occasion, it may be beneficial to have a GUI based tool for FTP/SCP file transfers where more than just a single (or couple) file needs to be copied to/from a remote system. For this, I would recommend FileZilla. Why?

  1. Multi-Platform - Available for Linux/MacOS/Windows
  2. OpenSource
  3. FREE

1. Great! How do I Install FileZilla on the Ubuntu Desktop?

Installation is pretty easy:

You can do it from a terminal using the following commands:

sudo apt update && sudo apt install -y filezilla

If you prefer the GUI method:

  1. Click Activities in the top left of your screen (Or the 9 Dots in the bottom left corner)
  2. In the Search Window, enter Ubuntu Software
  3. Click the Ubuntu Software
  1. When the Ubuntu Software window loads up, click the Magnifying Glass in the far top-left corner
  2. Enter filezilla in the search box
  3. Click the FileZilla entry in the results
  4. Click the Install button on the FileZilla page (not shown)

2. Launch FileZilla

After you have installed the software, when you click the "Activities"/9-dots menu and search for FileZilla, you will see it available to launch.

3. Add to Dock (optional)

For tools/applications that you use on a regular basis, you may want to pin the icon to the dock. Follow these two steps to do that:

  1. Right-Click on the FileZilla icon in the dock
  2. Click Add to Favorites

When you Quit the application, you will see that the FZ icon remains in the dock for quick/easy access
