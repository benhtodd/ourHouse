Solution: Understanding the Placement Decisions

Come on, give it another try before reading the solution!!!

Navigate to Infrastructure > Projects, open the Project configuration and navigate to the Provisioning tab.

Our DEV environment doesn't have storage profiles that fit the blueprint constraint requirement: 'cas.storage.profile.cap:pci'.  This leaves TEST and PROD as options, both of which have Storage Profiles that have the pci capability requirement above.

All deployments will land in TEST, and not PROD because the TEST regions have a higher priority set in the Project.
