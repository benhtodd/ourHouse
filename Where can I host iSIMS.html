iSIMS can be placed on our shared hosting webserver that hosts https://Livefire.solutions . As long as you follow the directions on this page, your iSIMs will be available at:

https://livefire.solutions/isims/[your-folder-name]

You can then have a lab manual,  power point, pdf, etc... point to that url in order to load the iSIM (Remember to make note that it will take a while for the iSIM to load due to its size.

Here's an example of the first iSIM being hosted on the site: https://livefire.solutions/isims/nsxcloud

How can I upload iSIMS?

You'll need to use FTPS to upload your files.

1. Recommended FTPS Client

A simple, free FTP Client is available for Windows, Linux, and Mac: FileZilla

2. FTP Client Configuration

  1. Configure your FTP Client to use the FTP Protocol
  2. Host should be set to: ftp.livefire.solutions
  3. Port is 21
  4. Encryption should be set to: Require explicit FTP over TLS -- DO NOT USE Unencrypted FTP
  5. You can get the user and password from our internal Confluence page: FTP Services \ Livefire.solutions (Requires SSO/VPN)

ONLY USE The specified ISIM FTP account!
